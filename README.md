# jQuery Wrapper

一個扔掉 jQuery 包袱的包裝器！  
A wrapper that throws away the jQuery baggage!

做法上可能不是最好，但會希望它能越來越好！  
The practice may not be the best, but I hope it to get better and better!

## Implemented function list

- Operation
  - each
  - find
  - show
  - hide
  - insertAdjacent (extended)
  - before
  - after
  - val
  - width
  - height
  - append
  - remove
  - eq
  - camelCase (extended)
  - css
  - offset
  - attr
  - prop
- Event
  - ready
  - on
  - off
  - keypress
- Ajax
  - $.ajax 
  - $.get
  - $.post
  - $.send (extended)
